Selektor pada CSS Inline
| No | Selektor | Fungsi                          |
| -- | -------- | ------------------------------- |
| 1  | `h1`     | Mengatur heading                |
| 2  | `p`      | Mengatur paragraf               |
| 3  | `div`    | Mengatur container/divisi       |
| 4  | `span`   | Mengatur teks kecil dalam baris |
| 5  | `button` | Mengatur tombol                 |
| 6  | `input`  | Mengatur form input             |
| 7  | `img`    | Mengatur gambar                 |
| 8  | `a`      | Mengatur link                   |
| 9  | `table`  | Mengatur tabel                  |
| 10 | `li`     | Mengatur item list              |

Selektor pada CSS Internal
| No | Selektor           | Fungsi                                          |
| -- | ------------------ | ----------------------------------------------- |
| 1  | `body`             | Mengatur seluruh halaman                        |
| 2  | `h2`               | Mengatur heading h2                             |
| 3  | `.kotak`           | Selektor class untuk elemen tertentu            |
| 4  | `#judul`           | Selektor id untuk elemen unik                   |
| 5  | `div p`            | Descendant selector untuk paragraf di dalam div |
| 6  | `ul li`            | Mengatur list di dalam unordered list           |
| 7  | `input[type=text]` | Attribute selector untuk input text             |
| 8  | `a:hover`          | class saat cursor diarahkan ke link             |
| 9  | `section`          | Mengatur elemen section                         |
| 10 | `*`                | Universal selector untuk semua elemen           |

Selektor pada CSS External
| No | Selektor       | Fungsi                           |
| -- | -------------- | -------------------------------- |
| 1  | `.container`   | Mengatur container utama         |
| 2  | `h1`           | Mengatur heading utama           |
| 3  | `.card`        | Mengatur tampilan card           |
| 4  | `#header`      | Mengatur elemen dengan id header |
| 5  | `nav a`        | Mengatur link di dalam nav       |
| 6  | `footer p`     | Mengatur paragraf di footer      |
| 7  | `button:hover` | Efek hover pada tombol           |
| 8  | `img`          | Mengatur gambar                  |
| 9  | `table tr`     | Mengatur baris tabel             |
| 10 | `ol li`        | Mengatur list pada ordered list  |
